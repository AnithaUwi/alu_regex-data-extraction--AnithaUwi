Employee TODO List Progress

This Python script fetches an employee's TODO list progress from a public REST API (jsonplaceholder.typicode.com) and displays the information in a specific format.

Project Description

This project contains a Python script that accepts an employee ID as input and uses the JSONPlaceholder API to fetch the employee's TODO list data. It then displays the progress of the employee's tasks: how many tasks are completed and their titles.

The script:
- Accepts an employee ID as a command-line argument.
- Fetches the employee's TODO list.
- Fetches the employee's name.
- Displays the progress in a readable format:
  - Employee name.
  - Number of completed tasks out of total tasks.
  - Titles of completed tasks.

Features

- Fetch employee data and task details from a public API.
- Display completed task details in a structured format.
- Error handling for API calls.
- Command-line interface for inputting the employee ID.
